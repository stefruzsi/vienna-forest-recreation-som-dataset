# vienna-forest-recreation-som-dataset
Cleaned dataset for SOM clustering analyses of forest recreation behaviour in Vienna metropolitan area.

This repository contains the cleaned dataset used for SOM clustering analyses of forest recreation behaviour in the Vienna Metropolitan Area, Austria.

## File
- `ForRest_SOM_dataset.xlsx` — cleaned SOM clustering dataset with variable information and codebooks

## Notes
This repository contains cleaned analytical data only. Direct identifiers are not publicly shared.
